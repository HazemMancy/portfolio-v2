Workbook Basis

  • Mass flowrate from volumetric flow and liquid density. If standard flow is selected, actual flow is corrected by standard / operating density.
  • Fluid velocity from actual volumetric flow and pipe internal area.
  • Pipe wave celerity uses elastic liquid/pipe correction: c = sqrt((K/ρ)/(1 + (K/E)(d/t)B)).
  • Maximum instantaneous surge pressure rise uses Joukowsky relation: ΔP = ρcv.
  • Pipe period uses tc = 2L/c.
  • Allowable surge pressure is a screening hoop-stress check using Ps = 2σt/(d + F·t).

Restraint Factor B

  • Configuration 1: B = 1 − μ/2.
  • Configuration 2: B = 1 − μ².
  • Configuration 3: B = 1.0.

Limitations

  • Screening-level liquid surge estimate only. It does not model valve closure time, pump trip, column separation, vapor cavity collapse, terrain profile, branch reflection, relief devices, or transient damping.
  • Final surge design requires project-approved transient analysis software and verified pipeline/mechanical data.
  • Use absolute pressure consistently for operating pressure; surge pressure rise is differential.
