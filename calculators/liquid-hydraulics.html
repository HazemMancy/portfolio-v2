Methodology

  • Darcy-Weisbach: ΔPf = f·(L/D)·(ρv²/2) — incompressible liquid
  • Turbulent (Re ≥ 4000): Swamee-Jain — f = 0.25/[log(ε/3.7D + 5.74/Re0.9)]²
  • Laminar (Re < 2300): f = 64/Re (Hagen-Poiseuille)
  • Transition (2300–4000): linear interpolation
  • Static: ΔPs = ρ·g·Δz / 105; +Δz = uphill
  • All calculations performed in SI internally; Imperial display applies conversion factors only

Unit Conversions (Imperial output)

  • Flow: 1 m³/h = 4.40287 USGPM
  • Length / ID / Δz: 1 m = 3.28084 ft; 1 mm = 0.03937 in
  • Density: 1 kg/m³ = 0.062428 lb/ft³
  • Pressure: 1 bar = 14.5038 psi
  • Head: 1 m = 3.28084 ft liq.
  • Velocity: 1 m/s = 3.28084 ft/s
  • ΔP/100 ft reported when Imperial selected

Assumptions

  • Steady-state, single-phase incompressible flow
  • Isothermal — ρ and μ constant along pipe
  • Newtonian fluid; major losses only (no fittings/bends)
  • Roughness default 0.0457 mm = 0.0018 in (commercial CS, GPSA Table 17-7)

Limits — API RP 14E

  • Screening-level guidance only — not hard code-compliance values
